SilvestreStock é uma plataforma **LegalTech Ambiental** desenvolvida para o controle, inventário e monitoramento de animais silvestres. Com um design focado na estética "Nature-Tech", o sistema une a potência do Bootstrap 5 com a elegância do Glassmorphism e CSS avançado.

## 🚀 Funcionalidades Principais

- **Bio-Inventory Dashboard:** Visualização em tempo real do plantel e status sanitário.
- **Geomonitoramento:** (Simulado) Rastreamento geográfico de espécies e habitats.
- **Controle de Suprimentos:** Gestão de medicamentos, alimentação e insumos biológicos.
- **Conformidade Ambiental:** Monitoramento visual de alertas e pendências regulatórias.

## 🛠️ Tecnologias Utilizadas

- **Estrutura:** HTML5 Semântico
- **Estilização:** Bootstrap 5, Custom Advanced CSS (Glassmorphism, Variáveis Dinâmicas)
- **Lógica:** JavaScript Vanilla (ES6+)
- **Iconografia:** Bootstrap Icons
- **Tipografia:** Google Fonts (Inter)

## 🎨 Design System: Nature-Tech

O projeto utiliza uma paleta de cores inspirada na biodiversidade:
- `Emerald Green`: Vida e crescimento.
- `Deep Moss`: Estabilidade e natureza.
- `Slate Blue`: Elementos hídricos e tecnologia.
- `Glassmorphism UI`: Camadas translúcidas que criam profundidade e foco.

## 📂 Estrutura do Projeto

```text
silvestre-stock/
├── css/
│   └── style.css       # Design System e Animações
├── js/
│   └── app.js         # Lógica de Dados e Interatividade
├── assets/            # Imagens e Recursos
└── index.html         # Aplicação Principal
```

## 📝 Como Executar

Simplesmente abra o arquivo `index.html` em qualquer navegador moderno. Não é necessário servidor backend para esta versão demonstrativa de alta fidelidade visual.
